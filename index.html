Installation

Follow the steps below to setup ChickBot in your server:

  1. Click the invite link at the TOP of this webpage.
  2. Select the server that you want to add ChickBot to, then click the Add to Server button. The default permissions should be:
    • Bot Permissions - Contains all of the permissions referenced
      • Permission - Bot
      • Permission - Administrator

List of Commands

Below is a list of all available commands.

    Moderation

      ?kick [user] [reason] - Kick a mentioned user.
      ?ban [user] [reason] - Ban a mentioned user.
      ?warn [user] [reason] - Warn a mentioned user.
      ?mute [user] [reason] - Mute a mentioned user.
      ?unmute [user] - Unmute a mentioned user.
      ?voicekick [user] - Kick a mentioned user from a voice channel.
      ?voicemute [user] - Mute a mentioned user in a voice channel.
      ?voiceunmute [user] - Unmute a mentioned user in a voice channel.
      ?deafen [user] - Deafen a mentioned user in a voice channel.
      ?undeafen [user] - Undeafen a mentioned user in a voice channel.

    Information

      ?userinfo [user] - Get Information about a mentioned user.
      ?serverinfo - Get information about the current server.
      ?botinfo - Get information about ChickBot.

    Utility

      ?setmodlog [channel] - Set a channel to receive moderation notifications.
      ?setmuterole [role] - Set a mute role for the mute command.
      ?setmodrole [role] - Set a moderator role for the server.
      ?prefix [prefix] - Give ChickBot a different command prefix.
      ?giverole [user] [role] - Give a role to a user.
      ?takerole [user] [role] - Take a role from a user.
      ?createrole [role_name] - Create a new role.
      ?deleterole [role] - Delete an existing role.
      ?createchannel [channel_name] - Create a new channel.
      ?deletechannel [channel] - Delete an existing channel.
      ?slowmode [channel] [duration (s)] - Set slowmode on a channel.
      ?purge [messages] - Purge a desired number of messages from a channel.
      ?mentionable [role] - Toggle the mentionability of a role.
      ?nickname [user] [name] - Give a user a nickname.
      ?embed [title] | [description] - Create an embedded message.
      ?lockdown [channel] - Put a channel into lockdown.

    Invite Logging

      ?invites - Get a list of your personal invite links with a number of how many times it has been used.
      ?invitechannel [channel] - Set a channel to receive invite messages when users join the server.
      ?togglejoin - Toggle the invite messages on/off.

    Leveling

      ?levellog [channel] - Set a channel to receive level up messages.
      ?mylevel - Check your own level and xp.
      ?checklevel [user] - Check another users level and xp.

    Miscellaneous

      ?invite - Get an invite link for ChickBot.
      ?support - Get a link to the ChickBot support server.
      ?bump - Bump your server in our server-bumps channel.
      ?vote - Vote for ChickBot.
